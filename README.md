# headline
模仿今日头条
界面是就是完全模仿的今日头条
数据也是在今日头条上面爬取的
基于flask+mysql
实现了登录，邮箱注册，点赞，发表微头条，发表评论，换头像，搜索文章，搜索用户等功能。
